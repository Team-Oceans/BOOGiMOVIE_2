# 📚 BooGiMovie
> **"책에서 시작된 감동이 스크린으로 이어지는 곳"** > 책·중고 도서·영화를 연결해주는 라이프스타일 콘텐츠 플랫폼

---

## 💡 프로젝트 기획 배경 (Background)
최근 인기 도서나 웹소설을 원작으로 한 영상 콘텐츠(영화, 드라마 등)가 제작되는 **'OSMU(One Source Multi-Use)'** 트렌드가 매우 강력해지고 있습니다. 

하지만 많은 사용자가 책을 읽은 뒤 영화를 찾아보거나, 영화의 여운을 이어가기 위해 원작을 구매할 때 각각 서로 다른 플랫폼을 전전해야 하는 번거로움을 겪고 있습니다. 

**BooGiMovie**는 이러한 미디어 간의 파편화된 경험을 하나로 묶기 위해 시작되었습니다. 사용자가 하나의 스토리 라인을 따라 도서와 영상을 유기적으로 탐색하고 구매할 수 있는 **'통합 미디어 소비 환경'**을 구축하는 것이 본 프로젝트의 핵심 목표입니다.

---

## 🛠 기술 스택 (Tech Stack)

### Backend
- **Language:** Java 17
- **Framework:** Spring Boot
- **Persistence:** Spring Data JPA
- **Database:** Oracle DB, Redis
- **Search & Logging:** ElasticSearch, Logstash, Kibana, Filebeat (ELK Stack)

### Frontend
- **Template Engine:** Thymeleaf
- **Language:** JavaScript (Vanilla JS), HTML5, CSS3

---

## ✨ 핵심 기능 (Key Features)

### 1. 통합 커머스 서비스
- **도서 및 영화 구매:** 다양한 카테고리의 도서와 영상 콘텐츠를 한 공간에서 조회하고 구매할 수 있습니다.
- **주문 관리:** 사용자의 구매 내역을 체계적으로 관리합니다.

### 2. ElasticSearch 기반 스마트 추천 시스템
- **연관 작품 큐레이션:** 단순한 키워드 검색을 넘어, ElasticSearch를 활용하여 도서의 원작/파생 영화를 매칭하거나 장르적 유사성이 높은 작품을 정교하게 추천합니다.
- **고도화된 검색:** 대량의 데이터 속에서도 빠르고 정확하게 원하는 도서나 영화를 찾을 수 있는 검색 기능을 제공합니다.

### 3. 실시간 로그 수집 및 분석
- **ELK Stack 활용:** Filebeat와 Logstash를 통해 서버 로그를 수집하고, Kibana를 통해 사용자 패턴이나 추천 시스템의 효율성을 시각화하여 모니터링합니다.
